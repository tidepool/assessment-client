TidePool Component Library

This page holds UI components so people can see how to implement various options of them and see how they look in one place.

`Proceed` ui_widget

This is a gui element that allows the user to indicate when he or she is ready to proceed, but does not block them from continuing their present activity.

Colors

These colors are used to build all colors in the interface. They are used as a base, and SASS functions like `transparentize` are used to make blends and variations.

`Share` ui_widget

This component abstracts away the differet share APIs and provides a common programatic and user interface for providing sharing functionality
